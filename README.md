## Agenda/To-Do Colaborativa

Aplicação web para gestão colaborativa de tarefas, desenvolvida no âmbito da unidade curricular de Engenharia de Software (ECGM, IPVC-ESTG).

**🔗 Aplicação online:** https://tasktide-uni.vercel.app/

---

## 📖 Sobre o projeto

Esta aplicação permite criar, gerir e acompanhar tarefas de forma simples e visual.
Foi desenvolvida com recurso a ferramentas de IA (vibe coding), seguindo um fluxo de trabalho colaborativo com GitHub, gestão Agile/Scrum e deploy automático via CI/CD.

### Funcionalidades principais

- Criar novas tarefas
- Marcar tarefas como concluídas
- Apagar tarefas
- Filtrar tarefas (todas / pendentes / concluídas)
- Persistência de dados no browser (localStorage)

---

## 🛠️ Tecnologias utilizadas

- **React** — biblioteca para construção da interface
- **Vite** — ferramenta de build e desenvolvimento
- **Tailwind CSS** — estilização da interface
- **Lovable** — geração de código assistida por IA (vibe coding)
- **GitHub** — versionamento e colaboração
- **GitHub Projects** — gestão Agile/Scrum (Product Backlog, Sprint Backlog, issues)
- **GitHub Actions** — pipeline de CI (lint e build automático)
- **Vercel** — deploy automático (CD)

## 🔄 Metodologia e processo de desenvolvimento

O projeto foi desenvolvido seguindo uma metodologia Agile/Scrum simplificada, organizada em 2 sprints:

- **Sprint 1** — Criação de site.
- **Sprint 2** — Fazer deploy do site no Vercel.
- **Sprint 3** - Relatório

A gestão de tarefas, backlog e issues foi feita através do **GitHub Projects**.
